When the async code is processing, it can result in resolve or reject state. Mean the request may be fulfilled or not.
If it is resolved, 'then' code will be executed.
If it is rejected, 'catch' code will be executed.

Promise always accept a function. Promise is a constructor fucntion so we write 'new' before it.

example:

var ans = new Promise((res, rej) => {
    if(true){
        return res();
    }
    else{
        return rej();
    }
})

ans
.then(function(){
    console.log('resolved');
})
.catch(function(){
    console.log('rejected');
})
